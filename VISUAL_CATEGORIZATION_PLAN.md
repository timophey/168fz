# План визуальной категоризации словарей

## Обзор

Цель: Визуально группировать словари по категориям на главной странице и в админ-панели с использованием цветовых схем и иконок.

## Решенные параметры

- **Страницы**: обе (главная + админ)
- **Тип группировки**: цветные карточки с иконками
- **Цветовая схема**: стандартная (красный/желтый/зеленый/синий и т.д.)
- **Иконки**: Font Awesome (уже подключен)

## Цветовая схема категорий

| Категория | Цвет (Bootstrap) | CSS Class | Иконка FA | Назначение |
|-----------|------------------|-----------|-----------|------------|
| Запрещенные слова | danger (red) | `bg-danger` | `fa-skull-crossbones` | Ст. 6.1 - ненормативная лексика |
| Иностранные слова | warning (yellow) | `bg-warning text-dark` | `fa-language` | Иностранные слова без аналога |
| Разрешенные иностранные термины | info (cyan) | `bg-info text-dark` | `fa-check-circle` | Список разрешенных терминов |
| Нормативные слова | success (green) | `bg-success` | `fa-book` | Стандартные русские слова |
| Технические термины | purple | `bg-purple` | `fa-code` | IT и технические термины |
| Топонимы | teal | `bg-teal` | `fa-map-marker-alt` | Географические названия |
| Аббревиатуры | orange | `bg-orange` | `fa-abbr` | Сокращения и аббревиатуры |
| Профессионализмы и жаргон | pink | `bg-pink` | `fa-users` | Профессиональный сленг |
| Термины/Тезаурус | secondary (gray) | `bg-secondary` | `fa-thesaurus` | Словари-ресурсы |

## Ожидаемый результат

- Визуальное разделение словарей по смысловым группам
- Улучшенная навигация и понимание структуры словарей
- Быстрая идентификация типа словаря по цвету и иконке
- Соответствие требованиям закона 168-ФЗ (приоритет - запрещенные и иностранные слова)

## Риски и ограничения

- Может потребоваться дополнительное место на экране (карточки занимают больше места чем таблица)
- Необходимо обеспечить адаптивность на мобильных устройствах
- При большом количестве категорий - сложность управления

## Альтернативные варианты (отклонены)

- Аккордеон: слишком скрывает информацию
- Вкладки: требует дополнительных кликов
- Только badges в таблице: недостаточно наглядно

## Критерии успеха

1. Все категории визуально различимы
2. Пользователь может быстро найти словарь нужного типа
3. Соответствие закону выделено наиболее ярко
4. Интерфейс остаётся удобным и не перегруженным
